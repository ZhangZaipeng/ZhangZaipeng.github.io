一、Linux命令基础
    1.Linux命令
        用于实现某一类功能的指令或程序
        命令的执行依赖于解释器程序（例如:/bin/bash）
    2.Linux命令的分类
        内部命令：属于Shell解释器的一部分
        外部命令：独立于Shell解释器之外的程序
    3.命令操作技巧
        反斜杠 "\":强制换行
        Ctrl + U :快速删除当前光标处之前的所有字符
        Ctrl + K :快速删除从当前光标处到行尾的所有字符
        Ctrl + L :快速清空当前屏幕中的显示内容
        Ctrl + C :取消当前命令行的编辑
二、ls和du命令
    1.ls(list) 详解
        -l : 以长格式显示文件和目录的列表
        -a : 显示所有子目录和文件
        -A : 与-a选项的作用基本类似，但有两个特殊隐藏目录不显示。"." 和 ".."
        -d : 显示目录本身的属性，而不是显示目录中的内容
        -h : 以更人性化的方式显示出目录或文件的大小
        -R : 以递归的方式显示指定目录及其子目录中的所有内容
    2.du(disk usage)命令详解
    用于统计指定目录所占用磁盘空间的大小,du命令常用的几个选项
        -a : 统计磁盘空间占用时包含所有的文件。而不仅仅只统计目录
        -h : 以更人性化的方式显示记录
        -s : 只统计所占空间总的大小

四、cp,rm和mv命令
    1.cp 复制文件或者目录
        cp [选项] 源文件或目录 目标文件或者目录
        -f : 覆盖目标同名文件或者目录时不进行提醒，而直接强制复制
        -i : 覆盖目标同名文件或者目录是提醒用户确认
        -p : 复制时保持源文件的权限、属主及时间标记等属性不变
        -r : 复制目录是必须使用此选项。表示递归复制所有文件及子目录
    2.rm 删除指定的文件或者目录
        rm [选项] 要删除的文件或者文件夹
        -f : 不提醒，直接强制删除
        -i : 提醒用户确认
        -r : 递归删除整个目录树
    3.mv 移动文件  或者  重命名
五、which和find命令
    1.which 查找Linux命令程序所在位置
        which 命令|程序名
        注：
            在PATH变量指定的路径中搜索某个系统命令的位置并且返回第一个搜索结果
            若在所有搜索路径中查找，加"-a"
            无法查找内部命令所对应的程序文件 ,如 which cd
    2.find 精细查找文件或者目录
        find [查找范围] [查找条件表达式]

        常用查找条件
            按名称查找 -name ：根据目标文件的名称进行查找，允许使用 (* ? 通配符) ?表示匹配一个 * 表示所有
                例如: find /etc -name "resol*.conf"
            按文件大小查找 -size ：根据目标文件的大小查找，一般使用 "+","-"
                例如: find /boot -size +10M  --> 大于10M的文件
            按文件属主查找 -user
            按文件类型查找 -type ：根据文件的类型进行查找，类型是 普通文件(f)、目录(d)、块设备文件(b)、字符设备文件(c)
                例如: find /boot -type d
            同时使用多个查找条件时 可使用 "-a" "-o"
                例如 find /boot -size +1M -a -name "vm*"
