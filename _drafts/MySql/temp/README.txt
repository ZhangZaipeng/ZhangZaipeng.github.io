违约规则：
  到期未付款计算滞纳金, 连续三天未还款平仓

  eg: 滞纳金=贷款金额*0.05%

计息规则：次日0点开始计算


还款规则：
  1.贷款周期 7天，15天，30天，60天，90天，120天

     < 30天 还款日 = 借款日 + 1 日
