1、eclipse
2、插件    hadoop-eclipse-plugin-2.6.0
3、复制编译的jar到eclipse插件目录中，重启eclipse
4、配置 hadoop安装目录
	window -> preference -> hadoop Map/Reduce -> Hadoop installtion directory （hadoop安装目录）
5、配置Map/Reduce视图
	window -> Open Perspective -> other -> Map/Reduce -> 点击 ok
	window -> show view -> other -> Map/Reduce Locations -> 点击 ok
6、 在“Map/Reduce Locations“Tab页 点击图标<大象+>或者在空白的地方放右键，选择”New Hadoop location“
	MR Mater 好DFS Master胚子必须好 mapred-site.xml和core-site.xml 等配置文件一致
7、打开Project Explorer  查看HDFS文件
8、建立 Map/Reduce任务   先启动Hadoop服务
	File -> New -> project ->Map/Reduce Project ->Next
